# Blog API

This project was bootstrapped with [NestJS]("http://nestjs.com/).

## Stack

- Node
- NestJS
- Express
- MobngoDB
- Mongoose
- Passport
- Bcrypt
- NestJS Command
- Class Validator + Transformer
- Sanitize HTML
- and more

### Installation

```bash
$ npm install
```

### DB Installation

Install MongoDB from [here]("https://www.mongodb.com/try/download/").

### Setup Environment Variable file

Create a `.env` file and populate it (see `.env.example`)

### Seed the Database

```bash
# seeds users and posts if none exists
$ npm run seed
```

### Running the app

```bash
# development
$ npm run start

# development:watch mode
$ npm run start:dev

```

### Build and run

```bash
# production
$ npm run build or yarn build

# production mode
$ npm run start:prod
```
